Router Help

Polyline Button: Allows the user to draw blockages on the map.

Point Button: Allows the user to select a location on the map. This location will be used when routes are calculated and will be the destination of the routes. Only one point can be chosen at a time.

Erase Button: Clears points and polylines

Selections Check Boxes: Allows the user to select which emergency responders are desired.

Analyze Button: Takes inputs and creates routes. Routes that are created are the shortest routes from responder locations to the point selected by the user while also accounting for created blockages.

Save Button: Saves output routes to a database that can be accessed using the "Saved Results" tab
